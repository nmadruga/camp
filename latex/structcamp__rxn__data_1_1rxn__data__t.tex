\doxysubsection{Detailed Description}
Abstract reaction data type. 

Time-\/invariant data related to a \mbox{\hyperlink{camp_rxn}{reaction}}. Types extending {\ttfamily \mbox{\hyperlink{structcamp__rxn__data_1_1rxn__data__t}{rxn\+\_\+data\+\_\+t}}} should represent specific reaction types, with unique rate equations. Extending types should not have data members, as these will not be passed to the child nodes after initialization. Instead all data required by an extending type during a model run should be packed into the condensed\+\_\+data arrays during initialization.
